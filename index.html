HTML Challenge I

Iteration 1- Inline vs block elements

Write a paragraph containing some span or em tags of different colors. Spans are iniline elements, meaning that default display property value is inline. Those elements will follow the horizontal flow of a document. Inline elements go next to another inline element so you won't have any line break until you decide do insert one (like here with a br tag).
Images are inline too ! That's a simple but super important point to remember ! Feel free to dig deeper with the MDN doc.
